# Publishing

This directory contains source-controlled publishing styles and runtime
configuration. The M1 pipeline uses:

- `styles.css` for standalone HTML
- `epub.css` for EPUB 3
- `puppeteer-config.json` for Mermaid rendering

Generated content belongs under the ignored root `dist/` directory. See the
[publishing pipeline specification](../specs/006-publishing-pipeline.md) and
[local development guide](../docs/05-operations/local-development.md).
