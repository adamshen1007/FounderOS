# M5B Hosted Platform Decision

## Decision

**No-go pending internal pilot evidence.**

## Sourced Facts

- M5A.2 provides local registration, allowlisted read-only external indexing,
  project details, governed jobs, recovery, backups, diagnostics, and
  pilot-record infrastructure.
- No completed pilot-session records exist at this decision point.

## Synthesis

Hosted accounts, collaboration, and billing would add permanent security and
operational scope before the local workflow has demonstrated repeated value.

## Reconsideration Gate

Reconsider through RFC-005 only after the pilot threshold in the M5A.1 protocol
is met. Until then, improve the local workflow from observed failures.

The M5A.3 status report automates threshold arithmetic only. A passing report
still requires human review of whether the projects are real, whether severe
security or accessibility findings remain, and whether hosted scope is
supported by observed need.
