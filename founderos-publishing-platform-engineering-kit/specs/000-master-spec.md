# FounderOS Publishing Platform Master Spec

## Goal

Create a reproducible publishing platform for founder books, playbooks, engineering kits, and company documentation.

## Non-goals

- Replace Git, Obsidian, or Notion.
- Depend on a single proprietary system as source of truth.
- Publish unsupported claims.

## Required Capabilities

- Markdown authoring
- Notion import/export
- Diagram generation
- PDF/EPUB/DOCX export
- Citation workflow
- Quarterly updates
- AI agent support
