# FounderOS Engine Interaction and Orchestration

The [engine interaction specification](engine-interaction.md) defines the M0
interaction model, human approval gates, failure rules, and the minimal
file-based workflow recommended for M1.

Review it before implementing the M1 Publishing Foundation. A complex runtime
must not replace the lightweight workflow until the manual and scripted path
has been validated.

The [agent artifact workflow](agent-artifact-workflow.md) defines the M4
proposal, verification, approval, and application handoff.
