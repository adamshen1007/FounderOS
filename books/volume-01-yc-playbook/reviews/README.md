# Volume 1 Review Records

This directory stores human review protocols and truthful review records for
the publication candidate. It contains no claim that a review occurred unless
the relevant file names the reviewer role, date, scope, findings, and outcome.

- `rights-and-brand-review.md` — rights, attribution, title, and independence
- `editorial-review-log.md` — substantive, evidence, and copy findings
- `beginner-beta-protocol.md` — representative-reader task protocol
- `accessibility-review.md` — HTML, EPUB, and DOCX accessibility record

Do not store private reader or company data. Use participant codes and
sanitized observations.
